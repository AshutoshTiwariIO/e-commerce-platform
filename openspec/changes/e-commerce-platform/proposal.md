## Why

Consumers increasingly shop online for electronics, but many platforms lack a focused, intuitive experience. This change builds a dedicated electronics e-commerce platform with modern UX and a Spring Boot backend.

## What Changes

- New React.js frontend with product catalog, cart, wishlist, checkout, and admin panel
- New Java Spring Boot REST API backend
- H2 in-memory database for development
- User registration and authentication
- Product browsing with category filtering and search
- Shopping cart management (add, remove, update quantities)
- Wishlist management
- Order placement with mock payment
- Order history for users
- Full admin panel for product and order management

## Capabilities

### New Capabilities
- `user-auth`: User registration, login, and session management
- `product-catalog`: Product listing, categories, search, and filtering (electronics focus)
- `shopping-cart`: Add/remove/update items in the cart
- `wishlist`: Save and manage wishlist items
- `checkout`: Order placement with mock payment processing
- `order-history`: View past orders and order details
- `admin-panel`: Admin dashboard for managing products, categories, and orders

### Modified Capabilities

None — this is a new project.

## Impact

- New React.js frontend application
- New Spring Boot backend application
- H2 database configuration
- No existing code affected (greenfield project)
